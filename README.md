# PixelNova Web Studio — Local Preview

A tiny, static landing page built with plain HTML, CSS and a small JS file.

## Files

- `index.html` — main page (provided HTML enhanced)
- `style.css` — modern, responsive styles
- `script.js` — theme toggle + small animations

## Run locally

You can open `index.html` directly in a browser, or serve with a simple HTTP server for best results:

Using Python 3:

```bash
cd /Users/ramesh/Desktop/PixelNova-Website
python3 -m http.server 8000
# then open http://localhost:8000
```

Or use the Live Server extension in VS Code.
